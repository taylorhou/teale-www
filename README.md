# teale-www

Website for [Teale](https://teale.com) — decentralized AI inference for everyone.

Run AI for free on your Mac, access the network for 100x less than OpenAI, or earn money sharing your idle compute.

## Structure

- `index.html` — Landing page
- `docs/` — Documentation site
- `stats/` — Live network statistics
- `install.sh` — Multi-platform install script
- `enterprise.html` — Enterprise page
- `privacy-policy.html` / `terms.html` — Legal

## Development

Static site hosted on GitHub Pages at [teale.com](https://teale.com). Just edit the HTML files and push.
